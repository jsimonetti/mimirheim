# Mimirheim (Beta)

This is the **edge channel** of Mimirheim. It is rebuilt automatically on
every push to the `main` branch of the source repository and may contain
incomplete or unstable features.

For stable use, install the **Mimirheim** add-on instead.

All configuration, options, and data directories are identical to the stable
add-on. See [DOCS.md in the stable add-on](../mimirheim/DOCS.md) for full
documentation.

The beta add-on uses the image tag `:edge`. Its configuration directory is
separate from the stable add-on, so both can run simultaneously without
interfering with each other.
